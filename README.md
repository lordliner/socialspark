## Getting Started

### Prerequisites
- Node.js >= 18
- MongoDB >= 6.0
- Python >= 3.8
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/socialspark.git
cd socialspark
```

2. Install dependencies:
```bash
# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install

# Install Python dependencies
cd ../ml-service
pip install -r requirements.txt
```

3. Set up environment variables:
```bash
cp .env.example .env
```

4. Start the development servers:
```bash
# Start frontend
cd frontend
npm run dev

# Start backend
cd ../backend
npm run dev

# Start ML service
cd ../ml-service
python app.py
```
